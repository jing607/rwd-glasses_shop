## Instructions

1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/your-repo.git
    ```
2. Navigate to the desired week's folder:
    ```sh
    cd your-repo/week1
    ```
3. Open `index.html` in your browser to view the web layout.
